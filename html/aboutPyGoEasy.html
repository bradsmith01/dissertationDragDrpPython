About PyGoEasy

PyGoEasy was part of Bradley Smith's Masters ofcomputer Science Dissertation project.
The project was titled 'Transforming Python; A Drag and Drop approach to Python'

The project had extensive research, with a focus on User Interface design to make it as easy to use as possible.
The research into Light and Dark theme User Interfaces, colours and contrast. Research also included building a website
that is accessible to people with different colourblindness and vision impairedness.

The layout of the website is meant to be as easy to use as possible without using the features left to right.
The user will use the Drag and Drop blocks first, then go to the right and click 'Convert To Python'.
Then the code is generated in the Code Editor.
