How To Use the IDE:

There are 4 key parts of the PyGoEasy Integrated Development Environment(IDE):

Click any to explore how to use them.

Toolbar
Drag and Drop Blocks
Code Editor
Practice Questions

ToolBar

The ToolBar is attached to the top of the IDE, and it has multiple features. Some of the features are for accessibility for the IDE, and others are just for helpful use. If you struggle reading the coloured syntax in the Code Editor, two features can help:
The 'Font Selector' and the 'Speak Text' button.

The 'Open Github' button will open Github in a new browser window. This is where you can sign in and save your progress if you wish. This can be achieved by making use of the 'Copy Code' button which will copy all existing code from the Code Editor ready for you to paste wherever you wish.

The 'Clear Editor can be used to clear the Code Editor.

! THIS CODE IS NOT SAVED ANYWHERE, SO IF YOU WOULD LIKE TO SAVE IT, DO SO BEFORE PRESSING!'

The most important button in the ToolBar is the 'Convert to Python' button, as indicated by the button being a different colour. When pressed, it will convert any changes you make in the Drag and Drop Block side, into the Code Editor as Python code.

Drag and Drop Blocks

The Drag and Drop Blocks part of the IDE is the fun part as there the code can be placed together throught them snapping together. If two blocks will not snap together it is likely they are not compatible. When the blocks snap together if you have your speaker on, there will be a clicking sound to indicate they have clicked together. If you chose the wrong block and do not want to use it.

The block can be dragged over the bin image and it will be deleted. Pressing the backspace when clicking on the chosen block will also delete it with the crumpling sound of paper.

The left side of the Block Drag and Drop section contains all the different types of blocks that can be found through the filtered names.

Code Editor

The Code Editor is editable and the code will be generated in there when the 'Convert To Python' button is pressed.

The different colours are to signify the different syntax types such as the functions, text, loops etc.

The numbers on the lines of the left side of the Code Editor are to help find errors in professional IDE. Numbered lines have been implemented here to help with the transition between the PyGoEasy IDE and a professional IDE.

Practice Questions

The 'Practice Questions' can be accessed by clicking the 'Practice Questions' button in the Popup Guide on the Home Page of the IDE. If you closed the guide and wish to revisit it, just click the 'Guide' button in the top Navigation Bar. The Practice Questions are hidden by default, but are visible once the button is pressed in the 'Guide'.
